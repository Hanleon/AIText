使用说明：
1. 此接口调用的是腾讯的文本内容安全接口，需要自己去申请账号和开通此功能后使用
2. 部署完成后打开页面，分别输入SecretId、SecretKey、BizType后发送检测即可
3. 返回数据可以显示AI分数，越低越好
